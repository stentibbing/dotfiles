# General-purpose context

This is my default, general-purpose context for `claude` sessions started outside of a specific project.

- I want explanations, analysis, and answers only.
- Do not create, edit, or delete any files, and do not run commands that modify the filesystem, git state, or any other system state.
- Read-only investigation (viewing files, running read-only commands) is fine when needed to answer a question accurately.
- If a task genuinely requires making changes, say so and ask me explicitly before doing anything, rather than assuming it's wanted.
